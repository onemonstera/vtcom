Alaska Aurora & Weather Dashboard

Multi-city aurora forecasting and weather comparison for an Alaska trip.

// the problem

Planning an Alaska trip where aurora viewing was a priority. The existing tools — NOAA's raw Kp index charts, the OVATION model map, and separate weather sites — each showed one piece of the puzzle. None of them answered the actual question: "Which city, which night, which hour gives me the best chance of seeing the aurora?"

// approach
  • Combined three data sources into one view: NOAA's 3-day Kp forecast (geomagnetic activity), NOAA's OVATION model (real-time aurora intensity by lat/lon), and Open-Meteo hourly cloud cover
  • Built a viewability score that combines Kp strength with cloud conditions — high Kp means nothing if the sky is overcast
  • Used nighttime-only cloud averages (8PM–6AM) instead of daily averages, since daytime clouds are irrelevant for aurora
  • Added an expandable 3-hour detail view matching NOAA's Kp intervals, so you can pick the exact window to go outside
  • OVATION "Right Now" card shows real-time aurora intensity per city using nearest grid point lookup with longitude conversion (negative west → 0-360)
  • Toggleable city filter lets you focus on just the cities you're actually visiting
  • Two-tier caching strategy: 1-hour TTL for weather/Kp data, 15-minute TTL for real-time OVATION data
// data pipeline
  • NOAA Planetary K-Index Forecast → 3-hour Kp windows, grouped by Alaska evening date, nighttime hours only (UTC 04:00–14:00)
  • NOAA OVATION Aurora Model → ~37,000 lat/lon grid points, nearest-neighbor lookup per city
  • Open-Meteo Hourly Forecast → cloud cover bucketed into 3-hour windows to match Kp intervals
  • All fetched client-side with localStorage caching — no backend needed
The aurora doesn't care about your schedule. The best you can do is stack the odds — right Kp, clear skies, right time window — and be ready to drive. This tool just makes the math visible.
// design decisions
  • Compact grid table with SVG cloud icons and color-coded star ratings — at-a-glance instead of paragraph-style forecasts
  • Kp bar visualization uses the standard NOAA green-to-red color scale
  • OVATION intensity displayed as discrete segment blocks — visual and proportional without needing numbers
  • Progressive disclosure: summary grid shows the 3-day overview, tap a day to drill into 3-hour windows
  • City filter persists in localStorage so it remembers your trip plan across sessions
